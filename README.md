# Virtual Plant API

## Descripción
API backend desarrollada en Go para simular la gestión y cuidado de plantas, estructurada mediante endpoints REST y lógica de negocio desacoplada.

## Tecnologías
- Go
- REST API

## Qué se hizo
- Diseño e implementación de endpoints REST para gestión de entidades.

- Estructuración de la lógica de negocio separando responsabilidades para facilitar mantenibilidad y escalabilidad.

- Modelado de un sistema backend orientado a simulación de estados y evolución de entidades.

## Qué aprendí
- Diseño de APIs backend en un lenguaje distinto al stack principal.

- Aplicación de principios de arquitectura y organización de código en entornos backend.

- Adaptación a nuevas tecnologías manteniendo buenas prácticas de desarrollo.
